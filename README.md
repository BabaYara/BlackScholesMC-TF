# MonteCarlo with Black-Scholes

It's just a proof of concept of using TensorFlow to compute quantitative finance problems
- Model: Black-Scholes
- Method: Monte Carlo
- Language: Python with TensorFlow

It does roughly 200 montecarlo by second
